Tags I have learnt in HTML:

Navigation tag -
The Navigation tag or "nav" tag, defines a set of navigation links.
Section tag -
The Section tag defines a section in a document. It also refers to a group of related content on a page. A section must have its own heading.
Image tag -
An Image tag or "img" tag is used to embed an image in an HTML page. The "img" tag has two required attributes - source ("src") and alternate text ("alt").
Ordered List tag -
The Ordered List tag or "ol" tag, defines an ordered list which can be numerical or alphabetical. The "li" tag or List tag, is used to define each list item.
Line Break tag -
The Line Break tag or "br" tag, creates a line break that is significant to the content. Useful in poems and addresses where the division of lines is important.
